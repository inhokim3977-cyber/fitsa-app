아래 사양대로 버튼 컴포넌트를 전면 개선해줘. 
파일 구조와 코드 수정은 네가 적절히 판단해서 style.css / index.html / (필요시) app.js에 반영해.

[브랜드 컬러(이미 적용된 테마와 동일하게 사용)]
- primary: #1E3D2B   (짙은 초록)
- secondary: #8B6A4A (우드 브라운)
- accent: #D9B377    (골드)
- bg: #F5F1EA        (아이보리)
- text: #222222      (텍스트)

[버튼 종류]
1) .btn-primary  : 기본 액션 (solid, 배경=primary, 글자=아이보리)
2) .btn-secondary: 보조 액션 (solid, 배경=secondary, 글자=아이보리)
3) .btn-outline  : 윤곽선 버튼 (테두리=primary, 배경=transparent, 글자=primary)
4) .btn-ghost    : 고스트 버튼 (배경=transparent, 글자=secondary, 호버만 살짝 배경)
5) .btn-danger   : 오류/삭제용 (배경=#B00020, 글자=#fff)  ※ 기본 제공만, 실제 사용은 선택

[크기 옵션]
- .btn-sm (height: 36px, padding: 0 12px, font-size: 14px, radius: 8px)
- .btn-md (height: 44px, padding: 0 16px, font-size: 15px, radius: 10px)  ← 기본
- .btn-lg (height: 52px, padding: 0 20px, font-size: 16px, radius: 12px)

[상태]
- hover: 명확한 대비(예: primary→accent 배경, 글자색은 primary로 반전)
- active: hover보다 5% 어둡게
- focus: 2px 외곽선 + 4px 외곽 그림자(ring) 색상=accent, 접근성 기준 준수
- disabled: opacity: 0.55, cursor: not-allowed, hover/active/focus 무시
- loading: 왼쪽에 스피너 아이콘 추가 후 텍스트 오른쪽 8px 여백

[아이콘]
- .btn > .icon-left, .icon-right 클래스 제공 (아이콘 크기 18px, 버튼 텍스트와 8px 간격)

[접근성]
- 텍스트 대비(배경 대비) WCAG AA 이상 확보
- 버튼 높이와 터치 타겟 최소 44px 권장(.btn-md 이상 사용)

[샘플 HTML (프리뷰 섹션 추가)]
- 페이지 하단에 미리보기 영역을 만들고 다음 버튼들을 나란히 렌더링해서 즉시 확인 가능하게 해줘:
  <button class="btn btn-primary btn-md">입어보기</button>
  <button class="btn btn-secondary btn-md">결과 저장</button>
  <button class="btn btn-outline btn-md">샘플 보기</button>
  <button class="btn btn-ghost btn-md">더 알아보기</button>
  <button class="btn btn-primary btn-md" disabled>처리 중</button>

[스피너 CSS]
- .spinner: width/height 16px, border 2px solid rgba(255,255,255,.6), border-top-color: #fff, border-radius: 50%, animation: spin 0.9s linear infinite
- @keyframes spin { to { transform: rotate(360deg); } }

[로딩 예시]
- .btn.loading 일 때 내부에 <span class="spinner"></span> + 텍스트 “처리 중…” 표시
- .btn.loading 은 disabled 스타일과 동일하게 포커스/클릭 비활성화

[코드 품질]
- CSS는 중복 최소화(공통 .btn 베이스 → 변형 클래스)
- transition: background-color, color, box-shadow 0.2s
- 다크모드는 이번 단계 제외(다음 단계로 분리)

위 사양을 반영한 CSS/HTML/JS를 생성·수정하고, 프리뷰 영역에서 상호작용(hover/focus/disabled/loading)까지 즉시 확인되게 만들어줘.
